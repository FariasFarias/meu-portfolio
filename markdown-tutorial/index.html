Hierarquia de títulos

Existem 6 níveis de títulos. Você pode usar # para definir os títulos. Basta apenas colocar quantidade de # em até seis de acordo com nível. Como no exemplo:

# Meu título de nível 1

Este é um exemplo de primeiro parágrafo.

## Meu título de nível 2

Este é um exemplo de segundo parágrafo.

### Meu título de nível 3

Este é um exemplo de terceiro parágrafo.

Ficando assim:

Meu título de nível 1

Este é um exemplo de primeiro parágrafo.

Meu título de nível 2

Este é um exemplo de segundo parágrafo.

Meu título de nível 3

Este é um exemplo de terceiro parágrafo.

Estilo de texto

Você pode usar estilos como: itálico, negrito e sublinhando.

Este texto está em _itálico_ ou *itálico*

Este texto está em __negrito__ ou **negrito**

Este texto é ***importante***

Este texto está ~~tachado~~

Texto <sub>sobrescrito</sub>

Texto <ins>sublinhado</ins>

Ficando assim:

Este texto está em itálico ou itálico

Este texto está em negrito ou negrito

Este texto é importante

Este texto está tachado

Texto sobrescrito

Texto sublinhado
